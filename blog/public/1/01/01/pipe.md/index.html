### Comandos Avançados do Git

Neste artigo, vamos explorar comandos que muitas vezes não usamos com frequência, mas que podem ser muito úteis no seu dia a dia como desenvolvedor. Vamos nessa!

Quando você usa o `git log`, ele mostra apenas o histórico dos commits, sem exibir as mudanças feitas. Se quiser ver as alterações de código diretamente, use a opção `-p` ou `â€“patch`. Isso vai gerar um diff que mostra o que foi adicionado (`+`) e removido (`-`).

```bash git log -p ```

```java public class HelloWorld { public static void main(String[] args) { System.out.println(“Hello, World!”); } } ```

Para encontrar commits que alteraram o número de ocorrências de uma string, use `-S`. Já para filtrar commits onde o diff contém linhas que correspondem a uma expressão regular, use `-G`.

```bash git log -S"System.out.println" git log -G"main" ```

Exemplo:

Se o commit mudou a linha que contém `System.out.println`, esse comando vai encontrar e exibir esse commit específico. O Git também permite o uso avançado de branches e o gerenciamento de mudanças temporárias com `stash`. Suponha que você está em uma branch de desenvolvimento, e quer criar uma nova branch para explorar uma nova feature:

```bash git checkout -b nova-feature ```

Você começa a fazer mudanças, mas então percebe que precisa resolver algo urgente na branch principal. Para salvar suas mudanças temporariamente, use o comando `stash`:

```bash git stash ```

Isso vai armazenar suas mudanças temporárias e deixar seu ambiente limpo para que você possa trocar de branch:

```bash git checkout main ```

Depois de resolver o problema e voltar à sua branch, você pode restaurar suas mudanças salvas com:

```bash git stash pop ```

Imagine que você começou a implementar uma nova funcionalidade:

```java public class Feature { public void newFeature() { System.out.println(“Implementando nova funcionalidadeâ€¦”); } } ```

Você faz o stash das suas mudanças, resolve o bug na branch principal, e então recupera o código salvo. Após usar `git stash pop`, seu código da nova feature será restaurado.
